# JavaFxRestClient
JavaFX 8 + JAX-RS Client APIで作ったRESTクライアントです。

## 使用技術
- JDK 1.8.0_60
- JavaFX 8
- JAX-RS Client API(jersey-client-2.22)
- Jackson 2.5.4

## 開発環境
- NetBeans 8.0.2
- JavaFX Scene Builder 8.0.0
- OS X El Capitan

## サーバー側
PayaraMicroRestServerプロジェクトを利用してください。

## 起動方法
IDEから、com.example.MainAppクラスから起動してください。（NetBeansで確認済み）
